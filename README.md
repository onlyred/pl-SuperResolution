## Dataset

### Flicker1024
[Link] (https://yingqianwang.github.io/Flickr1024/)

lickr1024 is a large-scale stereo image dataset which consists of 1024 high-quality image pairs and covers diverse senarios.

- Train : 1600
- Validation : 224
- Test : 224
